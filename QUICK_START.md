## 🔧 故障排除

### 问题 1: 样式未生效

**症状**: 页面显示，但没有 Web3 风格

**解决**:
1. 检查浏览器控制台是否有 CSS 加载错误
2. 强制刷新浏览器：`Cmd+Shift+R` (Mac) 或 `Ctrl+Shift+R` (Windows)
3. 清除浏览器缓存

### 问题 2: 端口被占用

**症状**: `npm run dev` 提示端口 3000 被占用

**解决**:
- Next.js 会自动使用下一个可用端口（3001, 3002...）
- 查看终端输出的实际端口号
- 或者手动指定端口：`PORT=3001 npm run dev`

### 问题 3: API 调用失败

**症状**: 点击 "Run Analysis" 后显示错误

**解决**:
1. 检查 `.env.local` 文件是否存在
2. 确认 `YOUTUBE_API_KEY` 已配置
3. 查看浏览器 Network 标签的 API 响应

### 问题 4: 恢复旧版 UI

**症状**: 想要恢复到旧版界面

**解决**:
```bash
cd "/Users/cancanguo/Desktop/BD KOL Tool/app"
cp page-old-backup.tsx page.tsx
```
